NEW MANDATORY FRAMEWORK: Realistic Ballistics (lib/realistic_ballistics.lua)

ALL weapon and tool mods MUST use this framework. No more inline Shoot/MakeHole/QueryShot/ApplyPlayerDamage chains.

What it does:
- Configurable per-weapon damage profiles (damage, pellets, spread, range)
- Custom distance falloff (full-power close range zone + exponential decay)
- Decoupled hole size vs penetration (Shoot + MakeHole)
- Uniform disk spread (sqrt distribution)
- GetPlayerAimInfo crosshair aiming
- 100% server-side, MP-safe

How to use:
1. Copy lib/realistic_ballistics.lua to mod's lib/ballistics.lua
2. #include "lib/ballistics.lua" in main.lua
3. Define profile: CreateBallisticsProfile({damage=28, pellets=24, spread=0.07, ...})
4. Fire: profile:FireFromTool(toolBody, muzzleOffset, player)

Full docs: docs/BALLISTICS_FRAMEWORK.md
Reference implementation: Hook_Shotgun/main.lua

This framework will expand to cover ALL tools (visual effects, audio, ammo, physics, non-weapon tools). Every tool should feel finely tuned and realistic.

New lint rules this session:
- DOUBLE-PROCESS-TIMER: catches timer desync (12 mods flagged)
- USE-SHOOT: catches old MakeHole damage pattern (12 mods flagged)
