API FINDING: Teardown has TWO input categories (per TEARDOWN_V2_API_REFERENCE.md):

PHYSICAL (raw keys, FAIL with player param): esc, tab, lmb, rmb, mmb, uparrow, downarrow, leftarrow, rightarrow, f1-f12, backspace, alt, delete, space, shift, ctrl, return, a-z, 0-9, mousedx, mousedy, mousewheel

LOGICAL (action keys, WORK with player param): up, down, left, right, interact, flashlight, jump, crouch, usetool, grab, handbrake, map, pause, camerax, cameray, extra0-extra6, etc.

Key insight: "up"/"down"/"left"/"right" are LOGICAL inputs (mapped to WASD by default), NOT arrow keys. Arrow keys are "uparrow"/"downarrow"/etc. This means Spells mod's InputDown("up", p) is CORRECT. Closed T169 as false alarm.
